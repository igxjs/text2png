# Sample Images

This folder contains example images demonstrating various text2png features.

## Generate Samples

Run the sample generator to create example images:

```bash
cd samples
node generate-samples.js
```

## Sample Images

- **test-avatar.png** - 200x200 avatar with centered initials
- **test-scaled.png** - Auto-scaled long text in fixed dimensions
- **test-fixed-width.png** - Fixed width with auto-calculated height
- **test-auto-size.png** - Auto-sized image (original behavior)
- **test-multiline.png** - Multi-line text with fixed dimensions

All samples demonstrate the flexible sizing capabilities of text2png.
